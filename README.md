Shoot-em-mini-project

# Nom du premier jeu: Tir aux canards

## Descirption du jeu
Le but du jeu est simple, il faut éliminer tous les canards pour remporter la partie !


## Usage
Compiler le programme avec make puis choisisser une difficulter en tapant make run\_easy , run\_mid ou run\_hard !
Pour personnaliser votre niveau, vous pouvez augmenter ou diminuer la taille de l'affichage ainsi que le nombre de canards
en changeant make run hauteur largeur nombre\_de\_canard.


## Difficultés rencontrées
On a eu des problèmes concernant l'affichage et comment les canards pouvaient être dirigés mais on a pu corriger le bug. 

# Nom du deuxième jeu : Tir aux canards 2: La revanche des canards

## Descirption du jeu
Le but du jeu est le même que le premier, mais les canards posent des tourelles pour détruire notre canon !

## Usage
Le principe est le même que le premier jeu avec des niveaux prédéfini make run\_easy  run\_mid  run\_hard

## Difficultés rencontrées
On a eu des problèmes pour les hitbox des tourelles qu'on a pas pu régler. Certaines tourelles peuvent apparaître au bord de l'affichage mais ce n'est pas trop génant.
